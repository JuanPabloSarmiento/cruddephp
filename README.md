Proyecto PHP - Entrega rápida
=============================
Contiene ejemplos de proyectos CRUD tomados de la guía:
 - Lista de tareas (tareas/)
 - Agenda de contactos (contactos/)
 - Inventario básico (inventario/)
Base de datos: crea tú la base de datos (por ejemplo `mi_primera_bd`) en MySQL.
Coloca estos archivos en htdocs (XAMPP) o en tu servidor PHP y edita config.php con tus credenciales.
